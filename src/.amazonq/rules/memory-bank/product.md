# Product Overview

## Purpose
iMAGE is a web-based gallery archive system designed for CVSU Carmona's creative organization to showcase, manage, and share collaborative artwork projects. It serves as a digital portfolio and content management platform for student artists and their creative works.

## Value Proposition
- Centralized platform for publishing and archiving creative collaborations
- Professional presentation of multimedia artworks (images and videos)
- Attribution system that properly credits multiple artists per project
- Easy sharing and discovery of creative works within the organization

## Key Features

### Artwork Management
- Upload and publish multimedia content (PNG, JPG, MP4)
- Support for multiple media files per artwork
- Rich metadata including title, overview, full content description
- Date tracking for artwork creation
- Category-based organization with custom collection support

### Artist Attribution System
- Multi-role artist credits (Graphic Artist, Writer, Videographer, Photographer)
- Multiple artists per role support
- Structured artist information display on each artwork card

### Gallery Dashboard
- Masonry grid layout for optimal visual presentation
- Real-time artwork count tracking
- Search functionality across all artworks
- Responsive design for mobile and desktop viewing
- Empty state guidance for new users

### Interactive Card System
- Image carousel with navigation controls
- Touch swipe support for mobile devices
- Expandable overview text with "more/less" toggle
- Edit and delete capabilities for artwork management
- Share functionality with QR code generation and link copying

### Sharing Capabilities
- Direct link sharing
- QR code generation for physical distribution
- Platform-adaptive sharing (bottom sheet on mobile, dropdown on desktop)

## Target Users
- Student artists and content creators at CVSU Carmona
- Organization administrators managing the gallery
- Viewers browsing the creative archive
- Collaborators working on team projects

## Use Cases
1. **Publishing Collaborative Projects**: Teams can upload their work with proper attribution to all contributors
2. **Portfolio Showcase**: Individual artists can display their contributions across multiple projects
3. **Event Documentation**: Capture and archive creative outputs from organizational events
4. **Content Discovery**: Browse and search through the organization's creative history
5. **External Sharing**: Generate shareable links and QR codes for promoting works outside the platform
